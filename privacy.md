# Libro.fm ALC Helper Privacy Policy

The Libro.fm ALC Helper extension processes your Libro.fm library and genre data entirely locally on your device to modify the appearance of Libro.fm ALC pages. 

No personal information, browsing history, or website data is ever collected, transmitted to external servers, or sold to third parties. All cached data is stored locally on your machine using Chrome Local Storage and can be cleared by the user at any time via the extension's Settings page.

**Support Data Export:** 
The extension includes a "Download Data for Support" button in the Settings. This feature solely generates a local `.json` file on your computer containing your cached data and extension error logs. This file is never transmitted automatically; it is only shared if you manually choose to attach it to an email or message to the developer for troubleshooting purposes.
